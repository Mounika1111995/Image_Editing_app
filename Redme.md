# Image Editing App with Streamlit & OpenCV

## Description
This project is a browser-based image editing application built using Streamlit and OpenCV. 
 Users can upload images, apply filters, and download edited images.

## Features
- Blur
- Sharpness
- Brightness
- Contrast
- Edge Detection
- Grayscale

## Setup Instructions
1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py
